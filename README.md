# ProyectoFinalUnidad1
Para la creacion de este sitio web se utilizó html como lenguaje principal, el cual permitio organizar el contenido mediante etiquetas como < header>, < nav >, < section >, < div>, y < footer>.

Se implemento un barra de navegacion que facilita el acceso entre una pagina principal llamada motos y el acceso a un segunda pagina
Tambien, se integró un carousel de imágenes para mejorar la presentacion visual del sitio, ademas se agregaron varias secciones de contenido las cuales incluyen listas, tablas formularios e inserción de videos

Para este diseño se utilizó, Bootstrap, que permitio trabajar con un sistema de columnas grip, para lograr un diseño responsive, adiccionalmente, se empleo CSS para personalizar estilos como colores, márgenes, tipografia y una buena distribucion de los elementos.
